```html
<p>Algum texto</p>
```

<p> é a tag de abertura
</p> é a tag de fechamento


## Elementos sem tag de fechamento
    
```html
<br>
<hr>
<img>
<input>
```

## Listas ordenadas

```html
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
```

## Listas não ordenadas

```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
```
